# ðŸ”¹ Types of Markup

## 1. Presentational Markup
- WYSIWYG - what you see is what you get
- Tells how text should **look** (bold, italic, color).  
- **Example:** LaTeX, old Word formatting, MS word ... etc

## 3. Procedural Markup
- Gives **details/instructions** to the computer on how to handle/transform text.  
- **Example:** XML processing commands.  

## 2. Descriptive (Semantic) Markup
- Tells what the text **means/its role** (title, paragraph, list).  
- **Example:** HTML, Markdown.
